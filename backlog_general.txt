# Liste de fonctionnalités, associées à leurs priorités :
--JEU--
FAIT 1- le joueur peut contrôler le personnage avec les touches du clavier
FAIT 2- il est possible de mettre le jeu sur pause
NON 2- le joueur dispose d'un temps limité pour gagner
NON 4- il est possible de sauvegarder les scores et les utilisateurs
FAIT 4- le jeu fait évoluer le héro dans plusieurs niveaux
NON 4- il est possible de gérer la vitesse du joueur

--HEROS--
FAIT 1- le personnage se déplace dans un labyrinthe
FAIT 1- le personnage ne peut pas traverser les murs
FAIT 1- le personnage gagne lorsqu'il trouve le trésor
FAIT 1- le personnage a une jauge de vie
FAIT 1- le personnage perd de la vie quand il est attaqué par un monstre
FAIT 2- le personnage peut attaquer les montres (grâce à une touche définie)
FAIT 3- le personnage perd de la vie dans les pièges
FAIT 3- le personnage gagne de la vie avec les potions


--MONSTRES--
EN PARTIE 1- les monstres se déplacent aléatoirement
FAIT 1- le monstre attaque le personnage lorsqu'il rentre à son contact
FAIT 2- les monstres ont une vie
NON 4- lorsque le personnage s'approche, le monstre ne se déplace plus aléatoirement et tente d'attaquer le héros

--LABYRINTHE--
FAIT 1- le labyrinthe est carré et constitué de murs
FAIT 3- le labyrinthe a des pièges
